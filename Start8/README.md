# Start8
attempt at styling the windows 10 (fullscreen) start screen to look like windows 8.

### IMPORTANT

this does not get your user picture, so you will need to provide a **User.png** file at "C:\Program Files\Windhawk\" or put an url in the **"Style constants"** for the user picture.

**same thing goes for the wallpaper**, if you dont have an **Wallpaper.png** file at "C:\Program Files\Windhawk\", you will get your desktop wallpaper without any opacity.

(also i couldn't make the app list selector behave like how it was originally on windows 8, so it will act as an vertical list as an way to compensate).
